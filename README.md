
# Crypto Client

Клиентская часть моего пет-проекта. Представляет из себя административную панель на Vue 3 + Tailwind + Flowbite 



## Installation

Установка стандартная

```bash
  npm install
  cp .env-example .env
  npm run dev
```
    
## Documentation

Реализовано:

* CRUD
* Авторизация через JWT токен
* Фильтры для поиска
* Пагинация

Что хотелось бы реализовать:
